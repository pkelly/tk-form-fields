This is a demo of the form fields for The Knot as designed by Silvia Tueros-Cossio. It was patternized and turned into this demo (and corresponding repo by Marco Carag).

Usage

  1. Download tk-form-fields.0.0.1.css.
  2. Place the file somewhere in your application that makes sense. a. In Rails, vendor/assets would probably make sense. b. In an Express Node app, try serving out of vendor -- eg: app.use('/static', express.static(__dirname + '/vendor'));
  3. Then architect your form's markup, following the below demo as an example. So long as you're using the appropriate classes and HTML, all should be well!

Contributing changes

DO NOT modify the tk-form-fields.0.0.1.css directly!!! Instead, fork the repo, make your changes there, and file a pull request. You could also file an issue

If you need to change things more urgently, override the css/html in your application's stylesheets and templates (eg, in a Rails app within app/assets/application.css).
